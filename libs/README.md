# Required JAR Files

This folder needs the following JAR files for the mod to compile:

## Download Instructions

1. **Waystones (Forge) 1.20.1-14.1.17**
   - Download from: https://www.curseforge.com/minecraft/mc-mods/waystones/files/5461158
   - Save as: `waystones-forge-1.20.1-14.1.17.jar`

2. **Balm (Forge) 1.20.1-7.3.9**
   - Download from: https://www.curseforge.com/minecraft/mc-mods/balm/files/5228813
   - Save as: `balm-forge-1.20.1-7.3.9.jar`

## After Downloading

Place both JAR files in this `libs/` folder:
```
libs/
  ├── waystones-forge-1.20.1-14.1.17.jar
  ├── balm-forge-1.20.1-7.3.9.jar
  └── README.md (this file)
```

Then commit these files to the repository so GitHub Actions can build successfully.
